### Pré-requis:

Lancez la commande suivante:
`npm install`

Une fois l'installation terminée, effectuez la commande suivante:
`npm install -g nodemon`

Puis lancer le serveur :
`nodemon server.js`
Par défaut, le serveur est accessible en local via le port 3000
